# Log Analysis Lab

## Overview
Performed log analysis using Splunk SIEM to identify 
suspicious activity and potential security threats.

## Tools Used
- Splunk Enterprise 10.2.2
- Kali Linux
- Auth logs (SSH authentication logs)

## What I Analyzed
- Failed SSH login attempts (brute force detection)
- Successful logins
- Suspicious IP addresses
- Invalid user login attempts

## Key Findings
- 7 failed login attempts detected (brute force attack)
- Suspicious IP 192.168.1.105 made repeated failed attempts
- Invalid user "hacker" probing for valid usernames
- 2 successful logins from legitimate users

## Skills Demonstrated
- SIEM tool installation and configuration
- Log ingestion and parsing
- Security event searching and filtering
- Threat identification and analysis

## Disclaimer
This lab was performed in an isolated virtual environment 
for educational purposes only.
